## 建议学习顺序

```
第 1 周：核心概念
  ├── 01 核心 Agent 循环（理解 ReAct 和 Function Calling）
  └── 02 工具系统架构（理解插件式设计）

第 2 周：外围系统
  ├── 03 Gateway 消息平台网关（理解多平台适配）
  └── 04 CLI 交互式终端系统（理解 TUI 设计）

第 3 周：智能特性
  ├── 05 记忆与学习系统（理解 RAG 和用户建模）
  └── 06 MCP/ACP 协议集成（理解协议设计）

第 4 周：深度优化
  ├── 07 Agent 内部子系统（理解 Prompt 工程和成本优化）
  └── 08 基础设施与研究工具（理解工程实践）
```
